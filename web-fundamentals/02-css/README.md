# CSS Folder Guide

Master selectors, layout systems, and motion.

## Order
1. selectors-specificity.css — Selectors, combinators, specificity, cascade
2. layout-systems.css — Flexbox, Grid, positioning, responsive patterns
3. animations-transforms.css — Transitions, keyframes, 2D/3D transforms

## Outcomes
- Write precise, maintainable styles
- Build responsive layouts with Flexbox/Grid
- Add performant motion and transitions

# React Folder Guide

Your main focus: fundamentals → hooks → component-patterns → advanced → performance.

## Order (Step-by-Step Learning Path)

### Foundation
1. **01-fundamentals.jsx** — Components, JSX syntax, props, state basics, rendering
   - **01-fundamentals-examples.jsx** — Code examples with output

### Intermediate
2. **02-hooks-essentials.jsx** — useState, useEffect, useContext, custom hooks
   - **02-hooks-examples.jsx** — Code examples with output

3. **03-component-patterns.jsx** — Composition, children, render props, HOCs, compound components
   - **03-patterns-examples.jsx** — Code examples with output

### Advanced
4. **04-advanced-concepts.jsx** — Context API, useReducer, error boundaries, portals, fragments
   - **04-advanced-examples.jsx** — Code examples with output

5. **05-performance.jsx** — useMemo, useCallback, React.memo, code splitting, lazy loading
   - **05-performance-examples.jsx** — Code examples with output

## Outcomes
- Strong foundation in React component architecture
- Confident with hooks and state management
- Deep understanding of component composition patterns
- Performance optimization techniques
- Real-world React development skills

## Prerequisites
- Complete TypeScript fundamentals (04-typescript/fundamentals.ts)
- Complete JavaScript async patterns (03-javascript/async-advanced.js)
- Basic understanding of Node.js and npm

## Time Estimate
- Total: 2-3 weeks
- Per file: 2-3 days each
- Practice projects: 1-2 weeks

## Key Concepts to Master
1. **Components**: Functions that return JSX
2. **JSX**: JavaScript XML syntax
3. **Props**: Passing data to components
4. **State**: Component internal data
5. **Hooks**: Functions to use state and effects
6. **Effects**: Side effects in functional components
7. **Context**: Global state management
8. **Performance**: Optimization techniques
9. **Composition**: Building with smaller components
10. **Patterns**: Common solutions to React problems

Understanding Semantic Tags

January 12, 2024 | By Developer

Common Semantic Tags

<header>
Represents introductory content or navigation aids at the top of a page or section
<nav>
Contains navigation links for major site navigation
<main>
Specifies the main content of the document (use once per page)
<article>
Represents self-contained, distributable content (blog post, news article)
<section>
Groups related content thematically
<aside>
Sidebar or supplementary content, indirectly related to the main content
<footer>
Contains footer information like copyright, links, or contact info
